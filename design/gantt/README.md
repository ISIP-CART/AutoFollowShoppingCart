# 甘特图说明

当前完整甘特图文件位于：

- `design/gantt/autofollowshoppingcart-day-gantt.xlsx`
- `design/gantt/autofollowshoppingcart-day-gantt-preview.png`

本版甘特图在原有日级排期基础上，新增了“关键路径任务”和“里程碑节点”的区分，用于表达哪些任务如果在对应节点之前没有完成，后续任务将无法继续开展。

## 颜色含义

- 蓝色：普通计划任务
- 橙色：关键路径任务
- 红色节点：里程碑节点，即“不完成则后续无法开展”的关键卡点
- 浅灰底：每周后两天的节奏参考

## 当前里程碑

- `M1` 基础链路跑通
  对应第 7 天前完成手动遥控、手机与下位机通信、底盘基础运动联调等基础链路任务。若未完成，后续自主跟随与安全联调无法开展。

- `M2` 自主跟随初版完成
  对应目标初始化、单人直线跟随和首轮参数调节完成。若未完成，后续目标丢失处理、安全策略和系统联调无法开展。

- `M3` 安全闭环联调完成
  对应急停、通信超时停车、目标丢失安全搜索与系统联调完成。若未完成，后续展示录制和最终交付无法开展。

- `M4` 最终展示闭环完成
  对应最终彩排与现场预案确认完成。若未完成，课程展示和材料收口无法按计划交付。

## 生成方式

甘特图由以下脚本生成：

- `design/gantt/gantt_builder.mjs`

如果后续需要继续调整任务排期、关键路径或里程碑颜色，请优先修改脚本后重新生成，而不是只手动改预览图。
