# design/background/ — 背景调研目录

## 目录用途

本目录存放自主跟随购物车项目的背景调研、资料摘要、来源链接和对本项目的启发。

所有调研服务于**首版课程原型**的目标收敛（4 周、3 人、室内低速安全跟随闭环），不把商业化智能购物车、完整 SLAM、自动结账等选做功能升级为必做功能。

## 编写原则

- 每份调研文档应明确：调研目的、关键事实、对本项目的启发、结论。
- 所有事实应标注来源（链接或文件路径），未确认的信息应标注"未确认"。
- 对首版不可行的技术路线应明确指出"首版不采用"及其原因。
- 文档之间可相互引用，但不重复展开已在其他文档中详述的内容。
